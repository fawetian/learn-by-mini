# GenericAgent 学习路线

目标：理解一个极简自主 agent 如何从小型种子运行时，逐步进化成可复用的技能系统，
然后用小实验重建这个机制。

以 `sources/generic-agent` 作为上游参考。第一轮源码阅读建议重点关注：

1. `agent_loop.py` 中的最小 agent 循环。
2. `llmcore.py` 中的 LLM 服务提供方抽象。
3. `agentmain.py` 中的核心运行时组装。
4. `assets/tools_schema*.json` 中的原子工具和结构定义。
5. `assets/sys_prompt*.txt` 中的系统提示词。
6. `memory/` 下的分层记忆。
7. 任务完成后的技能或 SOP 固化。
8. 浏览器、终端、文件、屏幕和 ADB 控制边界。
9. `frontends/` 下的前端适配器。
10. 安全、权限和用户确认机制。

初始章节规划：

1. 100 行 agent 循环。
2. 为什么原子工具可以替代大量专用工具。
3. 工具结构定义、分发和观察结果格式。
4. 分层记忆：规则、事实、技能和会话。
5. 把成功任务轨迹转成可复用技能。
6. 在相似任务中召回技能。
7. 自进化机制与插件生态的差异。
8. 构建一个微型自进化 agent 实验。
